
>简介

这是一个基于Vue2.0，面向移动端的仿饿了么的demo。
>技术栈

Vue+vue-router+vue-resource+ES6
>概述

使用了Vue进行页面骨架开发，使用vue-router进行前端路由切换，使用vue-resource进行数据获取。使用express搭建后台服务。使用eslint作为js代码风格检测。使用scss作为ss预处理工具。
